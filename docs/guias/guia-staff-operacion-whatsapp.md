# Guía para el equipo Staff: uso diario de WhatsApp y reservas

Esta guía está pensada para el personal del restaurante que usa el panel durante la operación diaria. No necesitas configurar Meta, tokens, contraseñas técnicas ni datos de WhatsApp Business. Esa parte corresponde al usuario Owner o a la persona responsable de administración.

Tu trabajo principal será revisar las reservas, atender cambios, consultar clientes y mantener la información del servicio al día para que WhatsApp y el equipo de sala trabajen con la misma agenda.

Fecha de la guía: 2026-06-30

## 1. Qué puedes hacer como Staff

Como usuario Staff, normalmente usarás el panel para:

- Revisar las reservas del día.
- Ver cuántos comensales están asignados.
- Consultar turnos pendientes.
- Marcar una reserva como sentada.
- Cancelar una reserva indicando el motivo.
- Marcar una reserva como no-show si el cliente no llega.
- Consultar el historial básico de clientes.
- Abrir WhatsApp, llamar o enviar email a un cliente cuando haya datos disponibles.
- Imprimir la lista de reservas del día.

Como Staff, no deberías modificar datos sensibles del restaurante, como tokens de WhatsApp, configuración avanzada de Meta o datos generales de administración.

## 2. Antes de empezar el turno

Al comenzar tu turno, abre el panel y entra con tu propio usuario.

Recomendaciones:

- No uses la cuenta de otra persona.
- No compartas tu contraseña.
- Si no puedes entrar, avisa al encargado o al Owner.
- Si ves datos extraños en la agenda, consulta antes de modificar reservas importantes.

## 3. Entrar al panel

1. Abre la dirección del panel del restaurante.
2. Escribe tu correo electrónico.
3. Escribe tu contraseña.
4. Selecciona o confirma el restaurante si el sistema lo pide.
5. Entra al panel.

Después de iniciar sesión, verás el menú lateral con las secciones disponibles. Las más importantes para Staff son:

- `Reservas`.
- `Clientes`.
- `Reportes`, si tu restaurante permite al equipo consultarlos.

## 4. Pantalla principal: Reservas

La sección más importante para Staff es `Reservas`.

Aquí verás la agenda del día y podrás controlar lo que ocurre durante el servicio.

En esta pantalla puedes encontrar:

- Total de cubiertos asignados.
- Turnos pendientes.
- Calendario o vista del día.
- Distribución de reservas.
- Lista de reservas con comensal, número de personas, horario y estado.
- Botón para imprimir la lista de hoy.

## 5. Revisar reservas al empezar el servicio

Antes de abrir sala o empezar el turno:

1. Entra en `Reservas`.
2. Revisa el número total de comensales.
3. Comprueba los turnos pendientes.
4. Mira las reservas confirmadas.
5. Revisa si hay muchas reservas concentradas a la misma hora.
6. Imprime la lista de hoy si el restaurante trabaja con hoja física.

Consejo práctico:

Si ves una reserva grande, un cliente frecuente o una hora con mucha carga, avisa al encargado de sala para organizar mejor las mesas.

## 6. Estados habituales de una reserva

En el panel puedes ver diferentes estados. Los más importantes son:

| Estado | Qué significa |
| --- | --- |
| Confirmada | La reserva está aceptada y debe prepararse |
| Pendiente | La reserva necesita revisión o confirmación |
| Completada | El cliente llegó y fue atendido |
| Cancelada | La reserva fue cancelada |
| No-show | El cliente no se presentó |

No cambies el estado de una reserva si no estás seguro. Un cambio incorrecto puede afectar la agenda, los reportes y el historial del cliente.

## 7. Marcar una reserva como sentada

Cuando el cliente llega y se sienta:

1. Busca la reserva en la lista.
2. Comprueba nombre, hora y número de personas.
3. Pulsa `Sentar`.
4. Verifica que la reserva quede actualizada.

Cuándo usarlo:

- Cuando el cliente ya está en el restaurante.
- Cuando la mesa ya fue asignada.
- Cuando el equipo de sala confirma que esa reserva está siendo atendida.

Evita marcar como sentada una reserva si el cliente solo llamó, escribió o todavía está esperando fuera.

## 8. Cancelar una reserva

Si el cliente cancela, es importante registrar el motivo.

Para cancelar:

1. Busca la reserva.
2. Pulsa `Cancelar`.
3. Elige el motivo más adecuado.
4. Escribe un detalle breve.
5. Confirma la cancelación.

Ejemplos de detalles correctos:

- `Cliente llamó para cancelar por enfermedad.`
- `Cliente escribió por WhatsApp y pidió cancelar.`
- `Reserva duplicada.`
- `Cliente cambió de plan y no desea reprogramar.`

Buenas prácticas:

- Escribe detalles claros y cortos.
- No uses comentarios personales o innecesarios.
- No borres información útil para el equipo.
- Si el cliente quiere cambiar de hora, consulta si conviene cancelar y crear una nueva reserva o pedir ayuda al encargado.

## 9. Marcar una reserva como no-show

Usa `No-show` cuando el cliente no se presenta.

Antes de marcar no-show:

1. Comprueba la hora de la reserva.
2. Espera el tiempo de cortesía definido por el restaurante.
3. Si el restaurante lo hace así, intenta contactar al cliente por teléfono o WhatsApp.
4. Si no hay respuesta o el cliente confirma que no vendrá, marca no-show.

Para marcar no-show:

1. Busca la reserva.
2. Pulsa `No-show`.
3. Elige el motivo correspondiente.
4. Añade un detalle breve.
5. Guarda la acción.

Ejemplos de detalles:

- `No llegó después de 20 minutos de cortesía.`
- `Se intentó llamar y no contestó.`
- `Cliente respondió por WhatsApp que no vendría.`

Importante:

No marques no-show demasiado pronto. Puede afectar el historial del cliente y los reportes del restaurante.

## 10. Atender mensajes o llamadas de clientes

Aunque WhatsApp pueda responder automáticamente, algunos clientes necesitarán atención humana.

Casos habituales:

- Quieren cambiar la hora.
- Quieren aumentar o reducir personas.
- Piden una mesa concreta.
- Tienen alergias o preferencias.
- Quieren cancelar.
- Tienen dudas sobre disponibilidad.

Cómo actuar:

1. Busca al cliente o la reserva en el panel.
2. Comprueba la información actual.
3. Si tienes permiso para hacer el cambio, actualiza la reserva.
4. Si no estás seguro, avisa al encargado.
5. Deja el motivo registrado cuando canceles o marques no-show.

## 11. Consultar clientes

Entra en `Clientes` cuando necesites revisar información de una persona.

Puedes buscar por:

- Nombre.
- Teléfono.
- Email.

En el perfil del cliente podrás ver:

- Datos de contacto.
- Historial reciente de reservas.
- Reservas completadas.
- Cancelaciones.
- No-show.
- Preferencias o notas de sala.
- Acciones rápidas como llamar, abrir WhatsApp o enviar email.

## 12. Cómo usar el historial del cliente

El historial ayuda a ofrecer mejor atención.

Ejemplos:

- Si el cliente suele venir a menudo, puedes avisar al encargado para darle una atención más personalizada.
- Si tiene notas de alergias, revisa esa información antes de sentarlo.
- Si tiene varios no-show, consulta antes de aceptar cambios delicados o reservas grandes.
- Si pidió una preferencia de mesa, intenta respetarla si la operación lo permite.

Importante:

Usa la información del cliente solo para el servicio del restaurante. No compartas datos personales fuera del equipo.

## 13. Usar WhatsApp desde el perfil del cliente

Si el cliente tiene teléfono registrado, puedes abrir WhatsApp desde su perfil.

Úsalo para:

- Confirmar una duda.
- Avisar de un cambio.
- Responder una consulta.
- Coordinar una llegada tardía.

Mensajes recomendados:

- `Hola, te escribimos del restaurante para confirmar tu reserva de hoy a las 21:00.`
- `Hola, hemos recibido tu solicitud. ¿Nos confirmas si la reserva sigue en pie?`
- `Hola, tu reserva está registrada para 4 personas a las 20:30.`

Evita:

- Mensajes demasiado informales si no conoces al cliente.
- Enviar datos internos del restaurante.
- Discutir con un cliente por WhatsApp.
- Compartir capturas del panel.

## 14. Imprimir la lista del día

En `Reservas`, usa `Imprimir Lista de Hoy` si el restaurante trabaja con papel en sala.

Antes de imprimir:

1. Revisa que estás viendo el día correcto.
2. Comprueba que la lista esté actualizada.
3. Revisa si hay reservas pendientes.
4. Imprime la lista.

Después de imprimir:

- Si entra una reserva nueva, avisa al equipo.
- Si se cancela una reserva, márcala también en el papel si lo usáis durante el servicio.
- No dependas solo del papel si el panel sigue recibiendo reservas por WhatsApp.

## 15. Qué hacer si WhatsApp confirma una reserva

Cuando WhatsApp confirma una reserva automáticamente:

1. La reserva debería aparecer en `Reservas`.
2. Revisa que el horario y número de personas tengan sentido.
3. Si hay alguna nota especial, comunícala al equipo de sala.
4. Si ves algo raro, avisa al encargado.

Ejemplos de situaciones a revisar:

- Reserva muy grande.
- Cliente pide una zona concreta.
- Reserva cerca de la hora de cierre.
- Cliente escribe varios mensajes cambiando datos.

## 16. Qué hacer si una reserva parece incorrecta

Si encuentras una reserva con datos incompletos o extraños:

1. No la borres sin consultar.
2. Revisa el teléfono del cliente.
3. Contacta al cliente si corresponde.
4. Avisa al encargado si puede afectar la sala.
5. Registra la acción correcta: cancelar, no-show o mantener confirmada.

Ejemplos:

- La reserva dice 20 personas pero el restaurante no acepta grupos tan grandes.
- El horario está cerca del cierre.
- El cliente escribió dos veces y parece haber dos reservas.
- El nombre no coincide, pero el teléfono sí.

## 17. Reportes para Staff

Si tienes acceso a `Reportes`, puedes revisar información básica de la operación.

Podrás ver:

- Reservas del periodo.
- Cubiertos.
- Asistencia.
- Cancelaciones.
- No-show.
- Reservas por día.

Uso recomendado:

- Revisa reportes si el encargado te lo pide.
- Úsalos para entender qué días tienen más carga.
- No tomes decisiones comerciales importantes sin consultar al Owner o al manager.

## 18. Buenas prácticas durante el servicio

Durante el servicio:

- Mantén la pantalla de `Reservas` actualizada.
- Marca como sentadas las reservas cuando el cliente llegue.
- Registra cancelaciones con motivo.
- Marca no-show solo cuando corresponda.
- Avisa al encargado si hay cambios grandes.
- Revisa los mensajes de clientes con calma.
- No prometas disponibilidad sin comprobar el panel.

## 19. Qué no deberías hacer como Staff

Evita estas acciones:

- Compartir tu usuario o contraseña.
- Copiar datos de clientes fuera del panel.
- Compartir capturas de reservas en grupos no autorizados.
- Cambiar estados de reservas sin motivo real.
- Marcar no-show antes del tiempo de cortesía.
- Cancelar reservas por error sin avisar.
- Tocar configuraciones de WhatsApp, Meta o tokens.
- Dar permisos a otros usuarios si no es tu responsabilidad.

## 21. Checklist durante el servicio

- [ ] Marco como sentadas las reservas que llegan.
- [ ] Registro cancelaciones con motivo.
- [ ] Marco no-show solo después del tiempo de cortesía.
- [ ] Reviso cambios de última hora.
- [ ] Confirmo datos antes de responder a un cliente.
- [ ] Aviso al encargado si hay una reserva complicada.

## 22. Checklist al cerrar el turno

- [ ] Revisé que las reservas atendidas estén actualizadas.
- [ ] Revisé cancelaciones pendientes de registrar.
- [ ] Revisé no-show del turno.
- [ ] Avisé al siguiente turno de reservas especiales.
- [ ] Cerré sesión si uso un ordenador compartido.

## 23. Ejemplos de respuestas útiles para clientes

Confirmación sencilla:

```text
Hola, tu reserva está confirmada para hoy a las 21:00 para 2 personas. ¡Te esperamos!
```

Consulta de cambio:

```text
Hola, revisamos disponibilidad y te confirmamos en unos minutos si podemos cambiar la hora.
```

Reserva fuera de horario:

```text
Hola, en ese horario no tenemos servicio. Podemos revisar otra hora disponible si lo deseas.
```

Llegada tarde:

```text
Hola, gracias por avisar. Te esperamos. Si el retraso supera el tiempo de cortesía, la mesa podría quedar sujeta a disponibilidad.
```

Cancelación recibida:

```text
Hola, hemos registrado la cancelación de tu reserva. Gracias por avisarnos.
```

## 24. Errores comunes y cómo evitarlos

| Situación | Qué puede pasar | Cómo evitarlo |
| --- | --- | --- |
| No marcar una reserva como sentada | El panel no refleja la ocupación real | Actualiza la reserva cuando el cliente llegue |
| Cancelar sin motivo | El equipo no sabrá qué ocurrió | Elige motivo y escribe un detalle breve |
| Marcar no-show demasiado pronto | Se perjudica el historial del cliente | Espera el tiempo de cortesía |
| Responder sin revisar disponibilidad | Puedes prometer una mesa que no existe | Mira siempre el panel antes de confirmar |
| Usar una cuenta compartida | No se sabe quién hizo cada cambio | Usa tu propio usuario |
| Ignorar notas de cliente | Puede perderse información importante | Revisa el perfil si hay dudas |

## 25. Resumen rápido

Como Staff, tu prioridad es mantener la operación clara y actualizada.

El orden diario recomendado es:

1. Entrar al panel con tu usuario.
2. Revisar `Reservas`.
3. Detectar reservas especiales.
4. Atender llegadas y cambios.
5. Marcar sentados, cancelaciones y no-show correctamente.
6. Consultar `Clientes` cuando necesites contexto.
7. Avisar al encargado ante cualquier caso dudoso.
8. Cerrar sesión al terminar si usas un equipo compartido.

Si mantienes la agenda actualizada, WhatsApp, sala y administración trabajarán con la misma información.
